# Getting and Cleaning Data Course Project

Daniel Ielpo

## Description
This file contains informations that describes how to run the code for the Course Project for the Johns Hopkins Getting and Cleaning Data Course.

## How to run the project
Put the script "run_analysis.R" in your working directory. 

When you run the script for the first time, it will create a subfolder './data' in the working directory to save the download data.

It downloads the data from the provided link, and then unzip the files into the 'data' subfolder.

After processing the files it will create data set as a txt file named "tidydata.txt" in the working directory with the result.
